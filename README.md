# Node_Message_Board

## 概要
授業の最終課題で制作したネット掲示板っぽいものです．Node.jsで書きます．

## 実装したい機能
- メッセージを入力し，投稿する機能

- 画像を投稿する機能

- ニックネームを登録し，投稿名にする機能

- 各ニックネームごとにメッセージを保存しておく機能

- ユーザ混在で新しい順にメッセージを表示できる機能 (某青い鳥SNSのタイムラインみたいな感じ)

- 24時間経つと再ログインを要求する機能 (セッションタイムアウト的な)

- パスワード認証機能

- パスワードのハッシュ化

- いいね機能
